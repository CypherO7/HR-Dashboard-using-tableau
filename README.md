## 📌 Project Overview

The **HR Analytics Dashboard** is a Tableau-based data visualization project created to analyze employee data and generate meaningful HR insights. This dashboard helps organizations monitor workforce trends, employee attrition, job satisfaction, and departmental performance through interactive visualizations.

The project focuses on transforming raw HR data into actionable insights for smarter and data-driven decision-making.

---

## 🚀 Key Features

✅ Employee Attrition Analysis  
✅ Workforce Demographics Insights  
✅ Department-wise Employee Distribution  
✅ Job Satisfaction Analysis  
✅ Salary & Experience Tracking  
✅ Interactive Tableau Visualizations  
✅ Professional & User-Friendly Dashboard Design
